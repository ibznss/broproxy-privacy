Политика конфиденциальности (BroProxy)

Мы НЕ собираем и не обрабатываем персональные данные пользователей.

Что используется

  • chrome.storage — локальное хранение настроек прокси на вашем устройстве.
  • proxy / webRequest — изменение системных прокси‑настроек для маршрутизации трафика (расширение не читает содержимое посещаемых сайтов).
  • api.ipify.org — публичный сервис для проверки текущего IP (один GET‑запрос, данные не сохраняются на стороне разработчика).
  • fonts.googleapis.com, fontawesome.com — статичные CSS‑шрифты и иконки для пользовательского интерфейса.

Что НЕ делается

  • Не отправляем данные на наши серверы.
  • Не отслеживаем историю посещения сайтов.
  • Не храним логи активности.
  • Не используем какие‑либо данные для рекламы или продажи третьим лицам.

Privacy Policy (BroProxy)

We do NOT collect or process any personal data of users.

What is used

  • chrome.storage – used only to store proxy settings locally on your device.
  • proxy / webRequest – used to change system proxy settings for routing traffic (the extension does not read the content of the websites you visit).
  • api.ipify.org – public service used to check the current IP address (a single GET request; no data is stored on the developer’s side).
  • fonts.googleapis.com, fontawesome.com – static CSS fonts and icons used for the user interface.

What we do NOT do

  • We do not send any data to our own servers.
  • We do not track your browsing history.
  • We do not store activity logs.
  • We do not use any data for advertising or selling to third parties.
